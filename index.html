About

I mostly work on the systems players don't see directly — data, progression, anti-cheat, currencies, rewards, things that have to keep working after 10,000 joins. I don't really enjoy rewriting everything or building giant frameworks. I'd rather fix the actual problem and keep the system simple than make it "scalable" for a scale that doesn't exist yet.

What I Build

Player Data

ProfileService saves, session locking, migration scripts when the schema changes.

Anti-Cheat

Speed/fly detection, teleport validation, movement sanity checks, instant flagging.

Economy

Passive income, prestige resets, daily rewards, gamepasses that actually persist correctly.

Server Logic

Ownership/possession tracking, hitbox detection, anything that has to be server-authoritative.

Networking

MemoryStoreService, RemoteEvents, keeping clients from being able to fake state.

Quests & Streaks

Attribute-based progress tracking that survives disconnects and bad timing.
